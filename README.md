# Usage
<!-- usage -->
```sh-session
$ npm install -g emzcli
$ emz COMMAND
running command...
$ emz (-v|--version|version)
emzcli/1.0.0 darwin-x64 node-v10.12.0
$ emz --help [COMMAND]
USAGE
  $ emz COMMAND
...
```
